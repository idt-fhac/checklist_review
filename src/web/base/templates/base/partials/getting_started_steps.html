{# Expandable setup steps; used by Help modal and workspace welcome modal #}
  • Declare LLM providers in config/providers.yaml at the repo root.
  • Set keys with env vars such as OPENAI_API_KEY, GOOGLE_API_KEY, or use local Ollama.
  • Without a configured provider, reviews and LLM-based PDF parsing cannot run.
  • Set pdf_metadata.method to rule_based or llm in config/providers.yaml.
  • LLM mode uses the provider referenced by defaults.pdf_metadata.
